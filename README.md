# 🎓 UniClub v3.1 - Студенческая платформа

Обновленная версия с правильной структурой университета.

## ✨ Что изменилось

### Структура университета
- **Иерархия:** Факультет → Направление → Группа
- Главный админ создаёт факультеты, направления и группы
- Всё связано между собой

### Регистрация студентов
- При регистрации студент выбирает:
  1. Факультет
  2. Направление (зависит от факультета)
  3. Группу (зависит от направления)
- После регистрации студент привязан к своей группе

### Роли и права
| Роль | Что может делать |
|------|------------------|
| **Главный админ** | Всё: факультеты, направления, группы, пользователи, клубы, мероприятия, расписание любой группы |
| **Староста** | Редактирует расписание своей группы + всё что может студент |
| **Админ клуба** | Управляет своим клубом и его мероприятиями |
| **Студент** | Просматривает расписание своей группы, подписывается на клубы |

### Расписание
- Староста видит и редактирует расписание ТОЛЬКО своей группы
- Студент видит расписание своей группы (без редактирования)
- Главный админ может выбрать любую группу и редактировать её расписание

## 🚀 Установка

### 1. База данных
Скопируйте содержимое `database-setup.sql` в SQL Editor в Supabase и выполните.

### 2. Запуск приложения
```bash
npm install
npm start
```
